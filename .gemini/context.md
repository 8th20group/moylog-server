# Moylog Project Context

## 📌 서비스 개요
- **목표**: 우아한테크코스 현재 크루와 미래 크루를 위한 모임 서비스 (`moylog`)
- **핵심 가치**: 사용자 경험(UX)을 고려한 세밀한 도메인 설계와 안정적인 인프라 구축

## 🎯 기술적 지향점
- **Pure Java & OOP**: 프레임워크에 의존하기보다 순수 자바 객체지향 원칙을 우선시함
- **Clean Architecture**: 도메인 로직이 외부 라이브러리나 프레임워크(Spring 등)에 오염되지 않도록 격리
- **TDD (Test Driven Development)**: 모든 도메인 로직은 단위 테스트로 검증하며, 테스트하기 쉬운 코드를 지향함
